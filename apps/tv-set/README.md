# TV Set App

## Description
The TV Set App is a creative, interactive portfolio experience designed to mimic a retro television set. Users can "change channels" to view different projects, sketches, and content, offering a unique and engaging way to explore my work.

## Tech Stack
- **Framework:** Next.js (v16+)
- **Library:** React 19
- **Creative Coding:** P5.js (`react-p5`)
- **Styling:** Tailwind CSS, Sass
- **Packages:** `@repo/channel`, `@repo/shell`
