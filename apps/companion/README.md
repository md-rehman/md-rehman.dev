# Companion App

## Description
The Companion App is prayer Tracker & Muslim Assistant app. 

## Screenshots
![Companion App](public/images/Companion.png)

## Tech Stack
- **Framework:** Next.js (v16+) with Turbopack for faster development
- **Library:** React 19
- **Styling:** Tailwind CSS (v4)
- **Authentication:** `@repo/auth` shared package
- **UI Components:** `@repo/atomic-ui`
