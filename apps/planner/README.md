# Planner App

## Description
The Planner App is a robust task management application heavily inspired by Trello. It allows users to visually organize their work using boards, lists, and cards, making project tracking intuitive and efficient.

## Tech Stack
- **Framework:** Next.js (v16+) with Turbopack
- **Library:** React 19
- **Authentication:** `@repo/auth`
- **UI Components:** `@repo/ui` shared package

## Upcoming Features
- Drag-and-drop interface enhancements for cards and lists.
- Real-time collaboration via WebSockets.
- Calendar view integration.
- Custom labels and due date notifications.
